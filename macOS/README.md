## About The Project

<div align="center">
  <figure>
    <img src="demo_macos.png" alt="Demo: macOS" height="500"/>
    <br />
    <figcaption>ၸႂ်ႉတိုဝ်းလွၵ်းမိုဝ်းပၢင်လူင် ၼိူဝ် macOS</figcaption>
  </figure>
</div>
<br />

တွၼ်ႈတႃႇၶူင်သၢင်ႈလွၵ်းမုဝ်းပေႃႉလိၵ်ႈၼိူဝ် macOS တႄႉလႆႈဝႃႈငၢႆႈသုတ်းၸႂ်ႉ program [Ukelele](https://software.sil.org/ukelele/) သေၶူင်သၢင်ႈဢွၵ်ႇ .bundle file ၵူၺ်းၵေႃႉၸႂ်ႉလႆႈယဝ်ႉ။

ယိူင်းဢၢၼ်းတႃႇ project ၼႆႉတႄႈၵေႃႈ တႃႇလဵပ်ႈႁဵၼ်းလႄႈတႃႇ tai-developer ဢၼ်သူၼ်ၸႂ်ၶႂ်ႈလဵပ်ႈႁဵၼ်း ဢမ်ႇၼၼ်သိုပ်ႇၶူင်သၢင်ႁႂ်ႈၶႅမ်ႉလႅပ်ႈၵႂႃႇတၢင်းၼႃႈၼၼ်ႉ လႆႈပိုၼ်ၽႄႈဝႆႉပၼ်ၶႃႈ။

## macOS Keyboard Layouts

ၶူင်သၢင်ႈ layout လူၺ်ႈ Ukelele ယဝ်ႉၸိုင် save .bundle file သေဝႆႉတီႈ ```/Users/${username}/Library/Keyboard Layouts``` ၵေႃႈၸႂ်ႉလႆႈယဝ်ႉ။
